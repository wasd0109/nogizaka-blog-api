# 乃木坂ブログ API

開発中、、、

# 目標

乃木坂 46 のブログを集めて、API で取得できるようにするプロジェクト。

# 内容

乃木坂 46 公式サイトからスクレイピングし、マークダウンやプレーンテキストで API から内容などを取得可能に。
